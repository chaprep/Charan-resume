Zebotech AI Product Pipeline

End-to-end AI content pipeline for an e-commerce electronics client. Scrapes competitor product data, runs it through a 3-stage Gemini AI generation pipeline, produces marketing-ready images, and publishes directly to Shopify — zero manual steps. Saving the client significant production hours weekly.

3

Pipeline Architecture
  • Stage 1 — Web scraping via Zyte API + Playwright: extracts and normalises competitor product data, bypasses bot detection
  • Stage 2 — Gemini AI: structured JSON prompts (versioned, model fallbacks) generate product descriptions with quality scoring
  • Stage 3 — Generative image production → direct publish to Shopify via GraphQL API
Internal Tooling Stack
  • FastAPI backend with SQLite-backed async task queue — no Redis needed, survives restarts
  • Next.js + TypeScript frontend UI — client's team runs the full pipeline with no engineering support
  • Docker Compose deployment on Hostinger VPS — live and maintained
  • Exponential backoff, concurrency controls, cascading failure detection

Algorithmic Options Trading System

Fully automated options trading system for Indian markets — built for personal use. Monitors support/resistance levels, executes trades atomically, manages trailing stop-losses, and mirrors trades across two accounts simultaneously. 7,000+ lines of Python.

Engineering Highlights
  • Atomic entry + stop-loss placement — succeeds or fails together, no partial state
  • Trailing SL state machine: NONE → ARMED → ARMED_BE → TRAILING
  • Dual-account mirror trading via parallel threads — both accounts execute simultaneously
  • Adaptive monitoring intervals: 30s normal, 5s near trigger levels
  • Daily state persistence — survives crashes and restarts without data loss
Instruments & Stack
  • 11 pre-configured instruments: NIFTY, BANKNIFTY, FINNIFTY + MCX commodities (CRUDEOIL, GOLD, SILVER)
  • Flask web UI for configuration, live monitoring, trade history and error log viewer
  • Dynamic expiry switching — weekly and monthly, auto-detected from broker API
